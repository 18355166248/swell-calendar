## 核心约定

- **宿主受控**：组件只派发交互意图与校验结果，事件数据始终以 `props.events` 为最终真值；
  宿主通过回调（`onEventCreate` / `onEventUpdate` / `onResourceVisibilityChange` …）决定是否更新。
- **统一字段**：`allDay` / `editable` / `resizable` / `resourceId(s)` / `timezone` / `recurrence`
  等字段贯穿各视图的校验链与渲染链。
- **失败回调**：`onEventCreateFailed` / `onEventUpdateFailed` 只表示本库同步规则校验失败，
  携带 `reason` + `policySource`，不代表宿主网络 / 服务端结果。
